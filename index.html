Instruções para rodar o backend localmente

1. Instalar dependências Python

Abra o terminal na pasta do projeto e execute:

pip install flask flask-cors pdfplumber spacy

Depois instale o modelo de linguagem (português ou inglês):

python -m spacy download pt_core_news_sm

2. Iniciar o servidor

Execute o arquivo app.py na mesma pasta:

python app.py

O servidor vai rodar em http://localhost:5000. Na primeira execução, o arquivo talentos.db é criado automaticamente. Deixe o terminal aberto enquanto usa o app.

3. Abrir o frontend

Abra este arquivo HTML no navegador normalmente (duplo clique ou via VSCode Live Server). O frontend vai se conectar automaticamente ao backend local.

O indicador no menu lateral mostra se o servidor está online.

Dica: Estrutura de arquivos

projeto/ ├── app.py ← backend Flask ├── index.html ← este arquivo └── talentos.db ← criado automaticamente (SQLite)

Os dados ficam no talentos.db — você pode copiar, fazer backup ou mover entre máquinas.
